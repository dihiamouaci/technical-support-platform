INSTALLATION DU PROJET GESTION TECH

1) Copier le dossier gestion_tech dans :
   C:\xampp\htdocs\

2) Lancer XAMPP puis démarrer :
   - Apache
   - MySQL

3) Ouvrir phpMyAdmin :
   http://localhost/phpmyadmin

4) Importer la base :
   - Cliquer sur Importer
   - Choisir le fichier gestion_tech_database.sql
   - Cliquer sur Exécuter

5) Lancer le projet :
   http://localhost/gestion_tech/
